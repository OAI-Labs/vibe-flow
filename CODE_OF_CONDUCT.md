# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

Be respectful. Disagree on technical grounds, not personal ones. Assume good
faith. Make this a project people want to contribute to.

## Reporting

If you experience or witness behaviour that violates the Contributor Covenant,
report it by opening a private email to the maintainers (see `SECURITY.md` for
the contact channel). All reports will be reviewed and kept confidential.

Maintainers may take any action they deem appropriate, including warnings,
temporary bans, or permanent bans from the project's spaces.
